# Ainyx App Graph Builder

Live Demo: https://app-graph-builder-zeta.vercel.app

GitHub Repository: https://github.com/misbaniha/app-graph-builder

## Features

* React Flow graph visualization
* Multi-application graph switching
* Node Inspector panel
* Runtime and Config tabs
* Value editing with Slider and Input sync
* Node deletion
* Fit View functionality
* Zustand state management
* TanStack Query data fetching
* Responsive dark-themed UI

## Tech Stack

* React
* TypeScript
* React Flow
* Zustand
* TanStack Query
* Tailwind CSS
* shadcn/ui
* Vite

## Local Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
